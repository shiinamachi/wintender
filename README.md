# wintender

Zebar Widgets Pack for Windows 11

The goal is to provide tray icon display and interaction, battery status, network status, and time display through the top bar, similar to macOS's top bar.

Since this is a project primarily for personal use, issues that are reported may not be resolved promptly.

## Features
- Show workspaces on komorebi
- Current time
- Tray icon display and pinned-tray icons (like Bartender on macOS)
- Network, audio, and battery status
- Right click context menu
  - Show task manager
  - Show zebar settings
  - Show wintender settings
- Customizable
